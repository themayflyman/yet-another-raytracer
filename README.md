# Yet Another Ray Tracer in Rust

Ray tracer implementation in Rust, based on [Ray Tracing in One Weekend Book Series](https://raytracing.github.io/).

## Featured:
  - Spectral color
  - Load and render .obj file
  - Multi-threaded Rendering
  - Static Dispatch
  - Code generation
